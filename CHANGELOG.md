# Changelog

Notable changes are documented here following [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-21

### Added

- Normalized MySQL schema for countries, coaches, athletes, teams, event entries, schedules, medals, and award auditing.
- Stored procedures and triggers for validated writes and audit capture.
- Five reusable analytical views and ten documented analysis queries.
- Fully synthetic Aurora Invitational dataset.
- Environment-configured Python CLI with parameterized queries and transaction handling.
- Unit, schema-contract, and MySQL invariant verification.
- Reproducible Docker Compose environment and GitHub Actions workflow.
